FAQ

What is it for?

This bot will assign you roles based on your WvW world. You will need a GW2 account API key.

What permissions are needed?

The API key only needs accounts permissions and must include exactly wvwbot in the name (lowercase).

How does it work

This bot saves your Discord ID and one API key for that account. You can replace the key by adding a new one. You can delete all data associated with a key by using the delete button above.

I transferred to another world. What now?

Your roles update when you order a new API key, and every 24 hours. You can press 'Sync now' to do it sooner. Your roles should update.

How can I add the bot to my server?

Add the Discord bot to your server.

'Login with Discord'? Are you trying to steal my password

Don't worry. You will see the bot requests your Discord ID and list of servers, which is what Discord will return. Your password stays with you and Discord.
